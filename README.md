Flashcards - manager flash cards
====

### This is app for educational purposes
## Setup

* `git clone https://github.com/surrexi/flashcards.git`
* `cd flashcards`
* `bundle install`
* `cp config/database.example.yml config/database.yml` and configure
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`

##Tech

* Ruby 2.2.4
* Ruby on Rails 5
